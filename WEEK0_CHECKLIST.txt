🆘 IF SOMETHING GOES WRONG
═════════════════════════════════════════════════════════════════════════════════

Can't extract frames:
  ☐ Check videos/raw/ exists
  ☐ Check video files are readable (try playing in VLC)
  ☐ Check free disk space (need 5+ GB)

Can't upload to Roboflow:
  ☐ Check internet connection
  ☐ Try uploading smaller batch (100 images)
  ☐ Check Roboflow website is up

Training is very slow:
  ☐ Check if using GPU: python -c "from ultralytics import YOLO; m = YOLO('yolov8m.pt')"
  ☐ If CUDA not available, reduce batch size in train_detector.py
  ☐ Reduce epochs (50 instead of 100)

Model accuracy is low:
  ☐ Need more annotated data (300+ minimum)
  ☐ Check annotation quality (tight bounding boxes)
  ☐ Retrain on more diverse footage
  ☐ Use larger model (yolov8l instead of m)

App crashes with detector:
  ☐ Check model path: models/yolov8_robots.pt exists
  ☐ Check CUDA compatibility (may need CPU mode)
  ☐ Try test inference first: python train_detector.py test VIDEOFILE
