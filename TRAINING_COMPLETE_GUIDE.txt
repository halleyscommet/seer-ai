WEEK 1 - TUESDAY-WEDNESDAY (Integration)
───────────────────────────────────────

Update your app to use trained model:

1. Edit app/robot_detector.py:
   Replace the placeholder implementation with:

   from ultralytics import YOLO
   from app.tracker import RobotDetection
   
   class RobotDetector:
       def __init__(self, model_path: str = "models/yolov8_robots.pt"):
           self.model = YOLO(model_path)
       
       def detect(self, frame, confidence_threshold: float = 0.5):
           results = self.model(frame, conf=confidence_threshold, verbose=False)
           detections = []
           
           for result in results:
               for box in result.boxes:
                   if box.conf[0] >= confidence_threshold:
                       x1, y1, x2, y2 = box.xyxy[0]
                       detections.append(RobotDetection(
                           x1=float(x1), y1=float(y1),
                           x2=float(x2), y2=float(y2),
                           confidence=float(box.conf[0])
                       ))
           
           return detections

2. Update app/ui.py:

TIPS FOR SUCCESS
═════════════════════════════════════════════════════════════════════════════════

Recording footage:
  • Film at least 5 different matches
  • Vary: times of day, lighting, team compositions
  • Position camera so bumpers are visible
  • Use tripod for steady footage
  • Avoid recording only your alliance

Annotation quality:
  • Draw tight bounding boxes
  • Include partial robots at edges
  • Be consistent in your labeling
  • Double-check difficult images
  • Don't skip too many frames

Training optimization:
  • Start with 300-500 annotated images
  • If accuracy is low, get more data
  • Train for more epochs if improving steadily
  • Use GPU if available (much faster)
  • Save checkpoints every 10 epochs

Troubleshooting:
  • Low accuracy? More training data needed
  • Slow training? Reduce batch size or epochs
  • Can't extract frames? Check videos exist
  • Can't annotate? Check Roboflow upload complete
  • Integration fails? Verify model path exists
